hugo + github pages

  1. 在linux mint上安装hugo 
    sudo apt-get install hugo
  2. 创建站点 
    hugo new site blog
  3. 使用主题 
    cd blog
git init
git submodule add https://github.com/RainerChiang/simpleness.git themes/simpleness

cp themes/simpleness/exampleSite/config.toml ./
cp -r themes/simpleness/exampleSite/content ./
  4. 启动 hugo server 看看效果 
    hugo server -D
  5. 构建静态网页 
    hugo -D
  6. 克隆username.github.io到本地 
    git clone https://github.com/yourusername/yourusername.github.io
  7. 把前面构建的静态网页copy到刚才克隆的库中 
    cd yourusername.github.io
cp ~/blog/public/* ./ -rf
  8. 提交 
    git add --all
git commit -m "hugo hosts on github pages"
git push -u origin main
  9. 浏览器中输入https://yourusername.github.io验证一下我们提交的东西是否已经发布成功
